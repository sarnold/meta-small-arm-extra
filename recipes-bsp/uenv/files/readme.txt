A Common uEnv.txt for iMX/6 and other Open Source Hardware on Mainline U-boot
=============================================================================

This is a set of uEnv.txt snippets for (mainly) imx6qdl machines.  We
override the vendor config with a mainline u-boot patch that follows
`LinuxOnArm`_ wiki and u-boot patches.


.. _: https://www.digikey.com/eewiki/display/linuxonarm/Home
